// Why CodingBar

A bill tells you what you spent. A dashboard tells you what you got.

Tokei and CodexBar show you a bill. CodingBar shows you a dashboard: what you and the AI got done, whether it was worth it, and how to spend smarter next time. Tokens and cost are the foundation; insight is the point.

git diff — bill → dashboard
-Tokens consumed
-Dollars spent
-A number with no context
+Lines, commits & files shipped
+Cost read as $/line and $/commit
+Where to spend smarter next time
+Quota burn-down before you hit a wall
// What it shows

// Install

Up and running in a minute.

install.sh
# 1 · Download the .dmg
$ open https://github.com/Gnonymous/CodingBar/releases
→ drag CodingBar.app to /Applications
# 2 · First launch (Gatekeeper)
$ xattr -dr com.apple.quarantine /Applications/CodingBar.app
# 3 · Or build from source
$ git clone https://github.com/Gnonymous/CodingBar
$ make run # pulse appears in the menu bar█
