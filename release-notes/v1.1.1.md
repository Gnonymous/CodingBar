- In-app auto-update via [Sparkle](https://sparkle-project.org). Off by default
  — open Settings and flip on "Auto-check for updates" to get a daily check and
  a one-click "Update now" button.
- Every update is verified against an EdDSA public key baked into the app
  before install, so a tampered feed or zip is silently rejected.
- After this manual upgrade, future versions install themselves — Gatekeeper
  only prompts on the very first manual install.
